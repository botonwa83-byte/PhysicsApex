常见问题

Q：购买了「完整功能解锁」，换新手机后怎么找回？

用同一个 Apple ID 登录新设备，打开应用进入解锁页，点击「恢复购买」即可。买断为一次性付费，永久有效，不会重复扣费。

Q：点了「恢复购买」却提示未找到记录？

请确认：①当前登录的 Apple ID 与购买时一致；②设备已联网；③系统「设置 → App Store」中已正常登录。仍无法恢复请邮件联系我们，并附上购买凭证（App Store 收据邮件）。

Q：内购为什么显示价格异常或无法购买？

通常是网络或 App Store 暂时性问题，请检查网络后稍候重试。若持续存在，请邮件告知你的设备型号与系统版本。

Q：免费版能用到什么程度？

免费开放部分题库、降维战例、模拟沙盘与武器，定律宇宙、每日一题、知识路线图等长期免费。解锁后获得全部题库、战例、三级重访与竞赛训练。

Q：训练会持续更新吗？

会。我们会持续补充题目、战例与模拟沙盘，已购用户的全部训练更新免费获得。

Q：数据安全吗？

所有学习数据仅保存在你的设备本地，我们不收集任何个人信息。详见 隐私政策。
